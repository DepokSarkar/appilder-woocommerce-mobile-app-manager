== Description ==

This Plugin is used to manage android and ios mobile app created from https://appilder.com/woocommerce/

Install our Appilder WooCommerce Mobile App Manager Plugin on your WooCommerce after you get your app. This plugin lets you manage the whole content of your app anytime and dynamically updates your app.

+ Navigation Menu : You can add items to your navigation menu. There is bunch of menu items you can add.
+ Widgets : You can make widgets that to be placed on your app home page or any other custom page you make.
+ Pages : Apart from the app home page, you could create any number of pages and fill them with widgets. You can add the page you make on app navigation menu.
+ Push Notification : Send your app uses custom notification anytime.
+ And much more features...

Install the plugin and start managing your app. Read more about the plugin usage on documentation.

https://appilder.com/woocommerce/docs
